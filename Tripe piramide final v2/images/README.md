# IMÁGENES REQUERIDAS - TRIPLE PIRÁMIDE DE ORO

## Imágenes que necesitas crear/subir:

### 1. **favicon.ico** (16x16px, 32x32px)
- Icono de la pirámide dorada
- Formato ICO con múltiples tamaños
- Colores: morado y dorado

### 2. **og-image.jpg** (1200x630px)
- Imagen para redes sociales
- Incluir logo y texto "Triple Pirámide de Oro"
- Diseño atractivo para compartir

### 3. **in-svg-blanco.svg** ✅ YA CREADO
- Logo de In Agencia para el footer
- Colores blanco/dorado

### 4. **hero-bg.jpg** (opcional)
- Imagen de fondo para el hero section
- Pirámides egipcias estilizadas
- Overlay morado translúcido

### 5. **pyramid-logo.png** (opcional)
- Logo 3D de la pirámide para el navbar
- Formato PNG con transparencia
- Tamaño: 100x100px aproximadamente

## Ubicación de archivos:
Todos los archivos de imagen van en la carpeta: `/images/`

## Especificaciones técnicas:
- **Formatos:** PNG, JPG, SVG, ICO
- **Optimización:** Comprimir para web
- **Responsive:** Preparar versiones para móvil si es necesario
- **Alt text:** Incluir texto alternativo descriptivo

## Nota:
Si no tienes las imágenes, el sitio funciona perfectamente sin ellas, usando elementos CSS y iconos de FontAwesome como fallbacks.
